Wizmini
Minimum OPDI slave implementation for AVR Atmega8 with less than 8 kB code size and about 500 bytes RAM consumption.
Uses communication via serial port (could be used with Bluetooth SPP adapter).

Project files for Programmer's Notepad 2. Compile with WinAVR-20100110.
You can use this project as a template for other AVR based implementations.
