OPDID Automation Server

OPDID, the Open Protocol for Device Interaction Daemon, is an open source, cross-platform automation server written in C++11 for performance and low memory footprint. It is designed specifically to run on smaller devices like the Raspberry Pi and can be used for home automation but also supports a variety of other usage scenarios. If you are looking for an affordable way to automate things then OPDID is probably worth a look.

Current version: 0.1.0 (prerelease) Download: Github repository
